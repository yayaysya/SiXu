# 学习路径：状态栏“进行中任务”展示 + 未完成任务并列

## 目标
- 在状态栏新增“进行中任务”行，显示：
  - 生成大纲阶段："学习路径 - 生成大纲中..."
  - 创建文件阶段："学习路径：正在创建 \"<文件标题>\" (i/total)"
- 当进入需要用户确认的阶段（学习地图预览弹窗）时，隐藏进行中行；未完成任务仍保持托盘可恢复。

## 实施要点
- 生成大纲：`src/views/combineView.ts` 在调用 `LearningPathGenerator.generateOutline` 前，通过 `StatusBarManager.showTaskStatus` 打开进行中任务；预览弹窗打开后 `hideTask` 隐藏。
- 创建文件：`src/learningPath/PathTaskQueue.ts` 在 for 循环内更新 `StatusBarManager.showTaskStatus`，显示当前创建的文件名；完成或异常时隐藏。
- 未完成任务：复用 `PendingTaskManager`；为状态栏托盘增加文本 "未完成任务 N"，保持点击弹出任务列表恢复。

## 受影响文件
- `src/views/combineView.ts`：新增 outline 阶段的进行中状态栏展示与隐藏
- `src/learningPath/PathTaskQueue.ts`：创建文件阶段进行中状态栏展示与隐藏
- `src/ui/PendingTaskManager.ts`：状态栏托盘显示文案（未完成任务 + 数量）
- `styles.css`：托盘标签样式

## 验证
1. 打开学习路径 → 生成大纲时，状态栏显示“学习路径 - 生成大纲中...”；弹出预览后行消失。
2. 确认创建 → 后台创建文件时，状态栏显示“正在创建 \"<文件>\" (i/total)”并实时更新；任务完成后行消失。
3. 关闭预览或结果弹窗 → 托盘显示未完成任务，可“继续”恢复。
