# Quiz 列表批量删除计划

## 上下文
- 任务：为学习中心试题列表提供批量删除与单项删除功能
- 目标：允许用户进入批量管理模式后物理删除选定的 Quiz Markdown 文件，执行前二次确认，删除后刷新列表
- 现状：`renderQuizListPage` 仅渲染列表和操作按钮，无删除能力

## 执行计划
1. 在 `src/views/combineView.ts` 增加批量模式及选中集合状态字段，并提供重置/退出辅助方法。
2. 更新 `renderQuizListPage`，增加批量管理切换与工具条逻辑，退出时清空选择。
3. 在 `renderQuizCardInLearning` 中根据批量模式渲染复选框，维护所选集合，并新增单项删除按钮。
4. 新增私有删除辅助方法，统一处理二次确认、文件删除与视图刷新（包括批量和单项）。
5. 更新 `styles.css` 为批量工具条和复选框添加样式以保持布局一致。
6. 手动运行构建验证（`npm run build`）确保无类型错误。
